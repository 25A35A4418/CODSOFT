🤖 RULE BASED CHATBOT USING PYTHON
🚀 This project was developed as part of my CodSoft Artificial Intelligence Internship (Task-01), where I built a simple Rule-Based Chatbot that responds to user queries using predefined rules and pattern matching techniques.
📌 Project Objective
The objective of this project is to create a chatbot that can understand user inputs and provide appropriate responses based on predefined conditions using Python.
📊 Project Description
The chatbot can handle common user interactions such as:
👋 Greetings (Hello, Hi, Good Morning)
❓ General Questions
📅 Date and Time Queries
🤝 Introduction Responses
🙏 Thank You Messages
👋 Exit Commands (Bye, Exit, Quit)
⚙️ Workflow
🔹 1. User Input Processing
Accepts text input from the user.
Converts input into a standardized format (lowercase).
🔹 2. Pattern Matching
Identifies keywords and phrases.
Matches user input with predefined rules.
🔹 3. Response Generation
Uses if-else conditions to generate suitable responses.
Provides instant replies based on detected patterns.
🔹 4. Conversation Handling
Maintains a simple conversation flow.
Allows users to continue chatting until they choose to exit.
🔹 5. Testing & Validation
Tested with multiple user inputs.
Verified response accuracy for different query types.
